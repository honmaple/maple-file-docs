Maple File

Multi-protocol cloud file management

Maple File is a server-free multi-protocol cloud file upload and management software that provides file upload, preview, operation, synchronous backup and other features

Download
Documents

Open Source

The complete source code of Maple File is hosted on Github. You can view and use it freely without worrying about the software being backdoored.

Cross-platform

Supports Web, Android, MacOS, Windows

Multi-protocol

Supports Local、S3、Webdav、FTP、SFTP、SMB、UPYUN、Alist、Mirror

File Preview

Support preview of video, audio, picture and text files

File Encrypt/Compress

Protect your privacy and prevent your files from being leaked or censored through AES encryption technology

File Sync/Backup

Support backup and synchronization between storages

Recycle Bin

Files accidentally deleted can be restored

Multilingual

Supports Chinese、English

Multiple Themes

Support multiple color theme switching
